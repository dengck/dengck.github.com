###安装Subversion

    $ sudo apt-get install subversion
    $ sudo apt-get install libapache2-svn

###Subversion常用命令

    svn add
    svn checkout (co)
    svn cleanup
    svn commit (ci)
    svn copy (cp)
    svn delete (del, remove, rm)
    svn diff (di)
    svn help (?, h)
    svn mkdir
    svn status (stat, st)
    svn update (up)
